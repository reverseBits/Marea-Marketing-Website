# Marea — Marketing Landing Page

Landing page for **Marea**, a menstrual cycle tracking app built for India. Science-backed. Privacy-first. Free forever.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main landing page |
| `privacy-policy.html` | Privacy policy |
| `data-deletion.html` | Data deletion request |

## Tech

Pure HTML/CSS/JS — no frameworks, no build step. Open any `.html` file directly in a browser.

## App

- Available on iOS and Android
- Built for India
